# Ферма Медовик - PRD

## Оригинальная задача
Создать адаптивный сайт интернет-магазина мёда и пчелопродуктов с категориями: Все, Мёд, Пчелопродукты, Настойки, Крема, Свечи, Аксессуары. При нажатии на товар - модальное окно с выбором граммовок и цен. Корзина с формой заказа через WhatsApp и Telegram. Админ-панель для управления товарами.

## User Personas
- **Покупатели**: люди, ищущие натуральный мёд и пчелопродукты
- **Администратор**: владелец магазина для управления каталогом

## Архитектура
- Frontend: React + Tailwind + Shadcn UI
- Backend: FastAPI
- Database: MongoDB
